# Egern 国家电网小组件

这是一个为 Egern 原生 Widget DSL 编写的小组件，不是 Scripting/Scriptable 脚本。支持主屏幕小、中、大尺寸与三种锁屏尺寸。

## 文件

- `state-grid.yaml`：Egern 模块配置
- `state-grid-widget.js`：小组件视图与数据兼容层
- `state-grid-capture.js`：网上国网响应数据本地采集器

## 安装

1. 把上面三个文件放进 Egern 可访问的同一个本地目录。
2. 在 Egern 打开“工具 → 模块 → +”，添加本地的 `state-grid.yaml`。
3. 开启模块并允许 `www.95598.cn` 的 MitM。若系统提示安装证书，请按 Egern 引导完成。
4. 保持 Egern 隧道开启，打开“网上国网”，依次进入电费余额、用电量/月度账单页面，让数据经过 Egern。
5. 回到 Egern 的“分析 → 小组件画廊”，选择“国家电网”预览。
6. 在 iOS 主屏幕添加 Egern 小组件，长按并选择“国家电网”。

## 多户和兜底配置

在模块的 Env 中设置 `ACCOUNT_INDEX`，第一个户号是 `0`，第二个是 `1`。如果所在省份接口结构暂未被自动识别，可以在 Env 中填写户号、余额、本月用电和本月电费，组件仍可正常展示。

`DATA_URL` 可读取兼容旧版网上国网小组件的数据数组，但不要使用会暴露明文账号或密码的第三方地址。此模块本身不会存储网上国网登录密码，也不会把采集数据发往第三方。

## 注意

- 网上国网各省接口存在差异，首次安装后应在网上国网 App 内实际打开相关数据页面。
- 电费和用电数据可能由电网延迟更新，小组件展示的是最近一次成功采集的缓存。
- 若 App 接口域名或字段变化，自动采集可能失效，此时可使用 Env 手动兜底。
